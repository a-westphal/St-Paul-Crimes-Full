Description of Tools

To create our application we used many different tools. First we created our own API to display data of Crimes from St. Paul. We used the Leaflet API in order to display our map of St. Paul. We also used the Nominatim API to convert between addresses and latitude/longitude to display our data. We used the framework Bootstrap to layout our page and get the different elements working together. We depended on other tools such as jquery, sqlite3, cors, and js2xmlparser in order to get our data and API working.

Interesting Findings

  1. The number of crimes that occurred in the Capitol River area was 909, this was around 450 more crimes than any other area
  2. Most neighborhoods have a little over 50% proactive police visits, but it seems around 75% of crimes in the Capitol River area are proactive police visits
  3. The neighborhood with the fewest number of crimes was Summit Hill with only 72
  4. There seemed to be an uptick in crimes in the Macalester-Groveland area on October 12th
  5. There was no serious change in number or types of crimes that occurred around Allianz Field on October 19th, 2019 for Tommie Johnnie
  6. There appears to have been more crimes on the weekends in the Summit/University area
